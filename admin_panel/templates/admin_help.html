Помощь по панели администратора

Обзор

На панели отображаются данные о производительности сервера в реальном времени. Здесь представлены три графика, которые отслеживают:

  • Запросы в минуту
  • Нагрузка на процессор
  • Нагрузка на оперативную память

Управление продуктами

Этот раздел позволяет управлять продуктами на платформе. Здесь вы можете добавить, редактировать, просматривать и удалять продукты:

  • Добавить новый продукт: Введите название продукта, цену, описание и URL изображения. Используйте форму для отправки данных и создания нового продукта.
  • Редактировать продукт: Выберите продукт по его ID и измените детали, такие как цена или описание. Сохраните изменения, отправив форму.
  • Удалить/Просмотреть продукт: Используйте ID продукта для удаления или просмотра информации о продукте.

Управление событиями

Управляйте событиями на платформе здесь. Вы можете:

  • Создать событие: Введите название события, дату, время и описание. Отправьте форму для создания нового события. Не вводите id чтобы создать новое событие.
  • Редактировать событие: Используйте ID существующего события для изменения его данных.
  • Удалить событие: Используйте ID события для его удаления из базы данных.

Логи

Страница логов предоставляет доступ к активности сервера. Здесь можно просмотреть системные события, ошибки и активность пользователей для контроля состояния системы.

Модерация

В разделе "Модерация" вы можете управлять участниками и их контентом. Сюда входят следующие функции:

  • Поиск пользователей по имени.
  • Редактирование профиля, почты, пароля и очков активности пользователя.
  • Изменение ролей пользователя: обычный пользователь, фанат, модератор, сотрудник, игрок или администратор.
  • Удаление пользователя или очистка его контента в случае необходимости.

Управление загрузкой файлов

Этот раздел позволяет загружать важные файлы (изображения, документы и т.д.) на сервер. Напишите путь файла на сервере и выберете файл и он сохранится или заменит существующий.

Выход

Используйте эту опцию для безопасного выхода из панели администратора после завершения работы.
